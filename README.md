# speechmote
A Speech-to-Emote translation Google Chrome Extension. Built using Python and JavaScript. Project for hackGT 8.

## Technologies Used
- Frontend (Extension)
    - JavaScript
    - RecordRTC
    - Browserify
    - Yarn
    - Node.js
- Backend (Model)
    - Python
    - FastAPI
    - Google Cloud Speech-to-Text
    - Google Cloud Natural Language
- Server (Deployment)
    - Docker
    - Google Cloud Compute Engine
